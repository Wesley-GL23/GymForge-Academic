# Exercícios Inferiores

Esta pasta contém GIFs demonstrativos de exercícios para membros inferiores.

## Requisitos dos GIFs
- Formato: GIF
- Tamanho máximo: 2MB
- Dimensões: 400x300px
- Duração: 3-5 segundos
- Loops: Infinito

## Lista de Exercícios
1. Agachamento (Squat)
2. Afundo (Lunge)
3. Elevação de Panturrilha
4. Agachamento Búlgaro
