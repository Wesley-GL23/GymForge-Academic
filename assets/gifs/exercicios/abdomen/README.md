# Exercícios Abdominais

Esta pasta contém GIFs demonstrativos de exercícios abdominais.

## Requisitos dos GIFs
- Formato: GIF
- Tamanho máximo: 2MB
- Dimensões: 400x300px
- Duração: 3-5 segundos
- Loops: Infinito

## Lista de Exercícios
1. Abdominal Tradicional
2. Prancha (Plank)
3. Elevação de Pernas
4. Abdominal Oblíquo
