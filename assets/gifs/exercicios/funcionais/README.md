# Exercícios Funcionais

Esta pasta contém GIFs demonstrativos de exercícios funcionais.

## Requisitos dos GIFs
- Formato: GIF
- Tamanho máximo: 2MB
- Dimensões: 400x300px
- Duração: 3-5 segundos
- Loops: Infinito

## Lista de Exercícios

2. Mountain Climber
3. Jumping Jack
4. Corrida Estacionária
