# Exercícios para Costas

Esta pasta contém GIFs demonstrativos de exercícios para as costas.

## Requisitos dos GIFs
- Formato: GIF
- Tamanho máximo: 2MB
- Dimensões: 400x300px
- Duração: 3-5 segundos
- Loops: Infinito

## Lista de Exercícios
1. Barra Fixa (Pull-up)
2. Remada
3. Extensão Lombar
4. Hiperextensão
