# Exercícios Superiores

Esta pasta contém GIFs demonstrativos de exercícios para membros superiores.

## Requisitos dos GIFs
- Formato: GIF
- Tamanho máximo: 2MB
- Dimensões: 400x300px
- Duração: 3-5 segundos
- Loops: Infinito

## Lista de Exercícios
1. Flexão de Braço (Push-up)
2. Extensão de Tríceps
3. Elevação Lateral
4. Desenvolvimento (Press)
